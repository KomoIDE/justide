Проект: Веб‑редактор с AI
Этот документ описывает простой и понятный план создания веб‑редактора, который работает через Cloudflare и использует AI для помощи пользователю.
1. Что мы создаём
Мы делаем веб‑приложение, где человек может редактировать файлы, получать подсказки от AI и управлять своими проектами.
2. Основные части проекта
• Фронтенд — интерфейс редактора и панели управления.
• Серверная логика — обработка запросов через Cloudflare Workers.
• База данных — хранение пользователей и проектов.
• Подключение AI — отправка запросов в Gemini.
• GitHub — хранение кода и автоматический деплой.
3. Как это будет работать
Пользователь открывает сайт. В редакторе он может создавать файлы, сохранять проект и общаться с AI. Вся логика проходит через Worker, а данные хранятся в D1 или KV.
4. Админ‑панель
Панель нужна для простого управления системой: просмотр пользователей, проектов, логов и настроек.
5. Основа для развития
Проект можно расширять: добавлять новые инструменты, улучшать редактор и подключать другие AI‑модели.
